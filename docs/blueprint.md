# **App Name**: Pharmacie Nouvelle d'Ivry Online

## Core Features:

- Product Catalog: Browse and search for health, beauty, hygiene, and baby products.
- Shopping Cart: Add products to a cart for purchase.
- User Account Management: Allow users to create and manage their accounts, addresses, and order history.
- Click & Collect: Enable users to place orders online and pick them up at the physical pharmacy location.
- Scan Ordonnance: Allow users to upload and submit their prescriptions for fulfillment.
- Personalized Product Recommendations: Utilize purchase history and browsing behavior as a tool to suggest relevant products powered by AI.
- Blog: Provide informative blog posts related to health and beauty.

## Style Guidelines:

- Primary color: Pharma blue (#1e88e5) for a clean and professional look.
- Accent color: Health green (#4caf50) for promotional badges and highlights, avoiding red.
- Background color: Light gray (#f8f9fc) for a minimalist feel.
- Body and headline font: 'Roboto', a sans-serif font, for a clean and readable experience.
- Use clear, simple icons for navigation and actions.
- Mobile-first responsive design with a grid-based layout for product listings and promotions.
- Subtle transitions and animations for UI elements and page transitions to enhance user experience.
